# node-demo
Node.js demo project

npm (Node Package Manager) is included in the Node.js distribution.

# How to update node and npm on Windows?

```
>node -v
v9.9.0

>npm -v
5.6.0

>curl -O https://nodejs.org/dist/v10.7.0/node-v10.7.0-x64.msi

>node-v10.7.0-x64.msi

>node -v
v10.7.0

>npm -v
6.1.0
```
